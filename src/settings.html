AI compression

Local Ollama is tried before the Fireworks companion for longer prompts. No API key is needed for local models.

Setup: install Ollama, run ollama pull gemma3:4b, then enable this option. Companion defaults to http://localhost:8000 (must match a host permission in the manifest, e.g. localhost:8000). Remote calls time out after 12s and fall back to local regex.

About BrevityPrompt

Purpose

BrevityPrompt automatically removes social fluff from your AI prompts on ChatGPT, Claude, and Gemini to:

  • Save context tokens
  • Increase signal-to-noise ratio
  • Reduce prompt noise

How It Works

  1. Click the extension toggle to enable BrevityPrompt
  2. Type your prompt as you normally would
  3. When you hit Enter or click Send, a preview appears
  4. Choose to send the shortened version or original

Privacy

✓ Regex shortening runs locally in your browser
✓ Local Ollama is optional and runs on your machine
✓ Fireworks compression is optional and prompts are never stored or logged by this extension

Default Categories

  • Greetings: Hi, Hello, Hey, Greetings
  • Politeness: Please, Kindly, Could you, I would appreciate if
  • Fillers: I was wondering if, I'm looking for, Basically, Essentially
  • Closings: Thanks, Thank you, Have a great day

Regex Tips

  • Use \\b for word boundaries
  • Use | to match alternatives: (option1|option2)
  • Use ? to make something optional: color?
  • Use .* to match any characters
  • Patterns are case-insensitive by default
